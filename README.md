O que é: Uma aplicação web interativa que gera senhas fortes de forma rápida e intuitiva.

Objetivo: Unir o desenvolvimento front-end à segurança digital, ensinando como a matemática protege nossos dados na internet.

Fundamento Matemático: Utiliza a análise combinatória (aumentando o número de caracteres e o tamanho da senha) e a entropia criptográfica (geração de números verdadeiramente aleatórios através da API crypto) para criar senhas impossíveis de serem adivinhadas por computadores maliciosos.

Tecnologias Utilizadas: * HTML: Estrutura e botões da página.

CSS: Visual moderno em estilo Dark Mode.

JavaScript: Lógica que calcula e sorteia os caracteres de forma segura.
